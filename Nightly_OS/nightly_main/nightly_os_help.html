Nightly OS User Guide

Disclaimer

This OS imitator web application has basic stuff, bugs may appear, and some tools would work in the future Python version! That one will have a backend code, and dynamic data handling!

Desktop

Click on the question mark to reach this user guide!

Click on the Start button to reach the settings, the file manager or shut down app!

Shut down will return you to the index.html file.

Click on the desktop with the right mouse button to open the dropdown list related to the OS imitator!

In case the dropdown list is not in the desired position, or covered by the browser's dropdown list, click with right mouse button somewhere else!

The new text, table, presentation file options will be in the future Python version, since these options are related to file management!

Taskbar size cannot be changed!

Icons' text can be edited, but the icons can't be drag-drop moved!

Double click the icons to start the program imitators or open windows!

The opened windows cannot be drag-drop moved, but the z-index/z-buffer focus can be changed by clicking on their names in the taskbar!

Computer

This icon will open a window, similar approach with Start button's list element File manager.

Double click on the Documents and the Images section to show files related to the folders of the web application!

The window minimize button on the top right corner will make the window disappear, but it will be available on the taskbar.

Click on the taskbar section of the window to make it reappear on the desktop!

The window restore button on the top right corner will make the window take the size of the desktop imitator.

Click on it again to resize the window to its original size!

The window close button on the top right corner closes the window and removes its taskbar section as well.

In case multiple windows are opened (this, the settings and the Aveomachine collection), click on the taskbar section to change z-index / z-buffer focus!

Settings

This window shows some stats, and a chance to change background of the desktop.

Click on the button related to the background to change the desktop's background!

Aveomachine collection

This opens a window with four links, each of them is related to one Aveomachine webgame.

Click on the button related to one game to open it and play with it online!

Text editor

This text editor imitator has 3 sections: text editor tools, editor box with 4 paragraphs, and font-family tools (from left to right).

Double click on one paragraph to set its focus to true! Only true focus value element can be edited!

Click on any button of the text editor tools to make change on the paragraph! In case you want to go back, hit Ctrl + Z keys!

Click on any of the font style buttons to change the font family of the focus wielding paragraph!

Changes can be made on the selected text area of the focus wielding paragraph!

Be aware: paragraph can be completely deleted!

Table editor

This table editor has a table of 26 columns and 13 rows. Alongside with a table, there are text editor tools and cell content changer functions.

Double click on one cell to set its focus to true! Only true focus value cell's content can be edited!

Click on any button of the text editor tools to make change on the cell's content! In case you want to go back, hit Ctrl + Z keys!

Changes can be made on the selected text area of the focus wielding cell!

Uppercase and Lowercase can be applied on text / string value wielding cells!

Round can be applied on numeric value wielding cells!

Use To-String and To-Num functions to convert the type of cell value before applying the forementioned functions!

Some of the functions are meant to be for dynamic input data handling, which will appear in the future Python version!

Be aware: cell content can be completely deleted!

Cell content can be drag-drop moved!

Presentation editor

This presentation editor has text editor tools, 3 slides in the editor interface and functions for animations.

Double click on one slide element (title, text, image) to set its focus to true! Only true focus value slide element can be edited!

Click on any button of the text editor tools to make change on the slide's content! In case you want to go back, hit Ctrl + Z keys!

Be aware: slide content can be completely deleted!

Click on any of the animation functions to try out an animation on the focus wielding slide element!

Linking multiple animations to play them one by one will appear in the future Python version!
