Click on the moving squares in order to prevent them from decreasing the health point value of the grey wall to zero.

Your unit types:

This is the wall. In case health point value gets zero, it will result game over.

Wall repairer unit. Adds health point value to the wall in case there is a decrease in it.

Sally forth unit. Strikes at one opponent unit if clicked.

Opponent unit types:

Regular opponent unit. Normal speed, 1 click is needed.

Big opponent unit. Normal speed, 4 clicks are needed.

Battering ram opponent unit. Big speed, 1 click is needed.

Boss opponent unit. Low speed, several clicks are needed. Comes as the last unit for the waves.

Wall improvement types:

Temple ( Building A )

Converts either regular or battering ram opponent units into player's units.

Work shop ( Building B )

Converts a player unit into a wall repairer unit.

Barrack ( Building C )

Converts a player unit into a sally forth unit.

About the buttons:

Regular

While this is the base mode, which can be used to click on opponent units, you have to click this button to switch back to this mode in case you are in either Temple, Work shop or Barrack mode.

Temple

You have to click this button to enter Temple mode, which with you can convert regular and battering ram opponent units into player units if you click them.

Work shop

You have to click this button to enter Work shop mode, which with you can convert player units into work shop units by clicking on the wall.

Barrack

You have to click this button to enter Barrack mode, which with you can convert player units into barrack units by clicking on the wall.
